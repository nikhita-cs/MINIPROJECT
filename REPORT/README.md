# 🍽 Smart Canteen Demand Prediction – Mini Project Report

## 📖 About the Project
This repository contains the Mini Project Report for the Smart Canteen Demand Prediction System.

The project focuses on predicting daily canteen food demand using Machine Learning techniques to reduce food wastage and improve planning efficiency.

---

## 🎯 Objective
To develop a predictive model that estimates daily canteen sales based on factors such as:
- Previous day sales
- Number of students present
- Exam schedule
- Weather conditions
- Day of the week

---

## 🧠 Methodology
The system uses Linear Regression to predict continuous sales values.  
Model performance is evaluated using R² Score and graphical analysis.

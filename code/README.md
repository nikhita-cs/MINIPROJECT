# 🍽 Smart Canteen Demand Prediction System

## 📖 Project Description
The Smart Canteen Demand Prediction System is a Machine Learning-based mini project developed to predict daily food sales in a college canteen. The system analyzes historical sales data and estimates the number of food items required for the next day.

This helps in reducing food wastage and improving inventory management.

---

## 🎯 Objective
To build a regression model that predicts daily canteen demand based on multiple influencing factors such as:
- Previous day sales
- Number of students present
- Exam period status
- Weather condition
- Day of the week

---

## 🧠 Machine Learning Approach
The project uses **Linear Regression** to predict continuous sales values.

The model was trained using historical dataset and evaluated using:
- R² Score
- Graphical comparison of actual vs predicted values

---

## 📊 Data Visualization
The following graphs are generated in the output:
- Actual vs Predicted Sales
- Sales Distribution
- Previous Day Sales vs Today Sales

These visualizations help in understanding model performance and data trends.
